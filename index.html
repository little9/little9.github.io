I'm a software developer at Data Curation Experts a software development company that specialiezs in digital repository software. Their main focus is Rails based solutions that use Hydra, Fedora, and Blacklight.

I'm currently the maintainer for the Ruby bagit gem.

Previously, I was the digital preservation and application development librarian at the University of Miami Libraries.

Some other open source projects that I have worked on include UM's APTrust Automation software which was used for testing the service before it went live, UDVD a customized Blacklight interface for browsing videos, and RAMP a tool for enhancing finding aids and publishing them on Wikipedia.

I also contributed to SubjectsPlus a tool for creating subject guides for libraries that can also be used as a library focused CMS.
